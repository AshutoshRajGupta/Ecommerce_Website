# ECOMMERCE WEBSITE
#### A FULLY MANAGED ECOMMERCE WEBSITE MADE WITH REACTJS AND STRIPE
#### USERS CAN EASILY REGISTER AND SIGNUP TO THE WEBSITE.
#### STRIPE PAYMENT METHOD IS USED FOR PAYMENT .

# TECHNOLOGIES USED
#### 1.REACTJS
#### 2.MATERIAL-UI
#### 3.NODEJS
#### 4.EXPRESSJS
#### 5.MONGODB
#### 6.STRIPE PAYMENT GATEWAY
